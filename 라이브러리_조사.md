## 기타 react 관련 정리

1. ### import

```js
import 객체, {객체} from [위치]
```

여기서 {}는 변수를 보내주는 방식의 차이  
{}는 export한 값을 가져올 때 사용
<br />

2. ### export

```javascript
export { 객체 };
```
